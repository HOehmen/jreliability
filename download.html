JReliability - Download and Source Code

JReliability has been formerly hosted on SourceForge, but has moved to GitHub.

Source Code

The source code of JReliability is available on GitHub. We encourage developers to join the project by forking and sending us pull requests; frequent contributors will be invited to permanently join the project team.

Download and Releases

JReliability is available as a Maven dependency. All releases are provided via Maven Central as well, so please use the download link on Maven Central to download the JAR (currently v1.3).
