JReliability - Download

Download latest release from SourceForge.net.

JReliability requires Java 5.0. The newest Java Runtime Environment is recommended.

Quick Start

  1. Download latest JReliability release.
  2. Unzip JReliability:
    unzip jreliability-{version}.zip
  3. Include jreliablity-{version}.jar in your project or try the Tester with:
    ./start.sh (Linux/Unix)
    start.bat (Windows)
    java -jar jreliability-{version}.jar (All Users)

SVN

The JReliability repository ist hosted by Sourceforge.net:
https://jreliability.svn.sourceforge.net/svnroot/jreliability

The following command will check out the whole project:
svn co https://jreliability.svn.sourceforge.net/svnroot/jreliability jreliability
If you only want to browse the svn, use the following link: SVN-Browse
